df — density fitting¶

This module provides methods to generate 3-center 2-electron AO integrals and the integral trasformation for 3-center 2-electron integrals.

Density fitting¶

Simple usage:

>>> from pyscf import gto, scf, df
>>> mol = gto.M(atom='N 0 0 0; N 0 0 1', basis='ccpvdz')
>>> mf = df.DF(mol).update(scf.RHF(mol)).run()
>>> mf = df.density_fit(scf.RHF(mol)).run()
